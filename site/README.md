# Nüfus Araştırması - İnteraktif Sunum

Türkiye'nin 81 ili için detaylı nüfus analizi ve modern sunum oluşturucu.

## Özellikler
- 81 il için detaylı nüfus verileri
- İnteraktif grafikler ve tablolar
- Modern siyah-mor neon tema
- Responsive tasarım
- İlçelere göre nüfus dağılışı

## Teknolojiler
- HTML5
- CSS3
- JavaScript
- Chart.js

## Kullanım
1. İl seçin
2. Otomatik analiz başlasın
3. Modern sunumu görüntüleyin
